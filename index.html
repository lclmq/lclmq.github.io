Mac环境下部署stf

  1. 下载安装node
  2. 下载homebrew, ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  3. brew install graphicsmagick zeromq protobuf yasm pkg-config
  4. 下载rethinkdb, http://download.rethinkdb.com/osx
  5. 下载stf，sudo npm install -g stf，可能会失败，推荐用淘宝镜像
  6. npm install -g cnpm --registry=https://registry.npm.taobao.org，然后 cnpm install -g stf
  7. 通过stf doctor检查需要的环境安装情况，可能会提升protobuf3.x版本不支持，通过brew重新安装2.6.1
  8. brew unlink protobuf, brew install protobuf260
  9. 运行rethinkdb,指定端口和缓冲大小启动：rethinkdb --bind all --cache-size 8192 --http-port 8090
  10. 启动stf，stf local 或者 stf local --public-ip xx.xx.xx.xx --allow-remote
  11. 本地在http://localhost:7100访问
  12. adb服务需下载jdk和sdk
