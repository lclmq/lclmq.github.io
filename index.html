Mac下openstf环境搭建

  1. 下载homebrew, ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  2. brew install graphicsmagick zeromq protobuf yasm pkg-config
  3. 安装thinkdb，http://download.rethinkdb.com/osx
  4. 安装stf，npm install -g stf(一般很慢，若失败，改用淘宝镜像)；
  5. 安装淘宝镜像，npm install -g cnpm --registry=https://registry.npm.taobao.org
  6. 使用cnpm install -g stf
  7. vim ~/.bashrc 添加如下信息： export PATH=$PATH:/usr/local/Cellar/protobuf/3.1.0/bin/ （请以本地下载路径为准） export PKG_CONFIG_PATH=/usr/local/Cellar/protobuf/3.1.0/lib/pkgconfig/ （请以本地下载路径为准） :wq (保存并退出)
  8. 使用stf doctor检查是否所有依赖环境正确安装，当前对protobuf的3.x版本不支持，所以需要通过brew下载指定2.6.1版本的protobuf
  9. brew unlink protobuf, brew install protobuf260
  10. 启动rethinkdb, rethinkdb --bind all --cache-size 8192 --http-port 8090， 可以通过localhost:8090查看数据库
  11. 启动stf, stf local, 或者 stf local --public-ip xx.xx.xx.xx --allow-remote
